Como Usar o Site

Este site permite que você criptografe e descriptografe mensagens usando o algoritmo AES (Advanced Encryption Standard). Veja como é simples:

Criptografar uma Mensagem:

  • No campo "Digite sua mensagem", escreva a mensagem que você deseja criptografar.
  • No campo "Digite sua chave secreta", insira uma chave secreta. Certifique-se de lembrar ou guardar essa chave.
  • Clique no botão "Criptografar com AES". O site irá criptografar sua mensagem e o resultado aparecerá no campo "Mensagem criptografada".
  • Para copiar a mensagem criptografada, clique no botão "Copiar Mensagem Criptografada".

Descriptografar uma Mensagem:

  • No campo "Mensagem criptografada", cole a mensagem criptografada que você deseja descriptografar.
  • No campo "Digite sua chave secreta", insira a mesma chave secreta que foi usada para criptografar a mensagem.
  • Clique no botão "Descriptografar com AES". Se a chave estiver correta, a mensagem original aparecerá no campo "Mensagem descriptografada".
  • Caso queira copiar a mensagem descriptografada, clique em "Copiar Mensagem Descriptografada".
